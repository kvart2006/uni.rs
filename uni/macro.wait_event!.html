Wait for an event to occur

This macro allows a thread to wait for a condition to become true. If the condition is false, the thread will block on the queue given as parameter. This macro MUST be called with local irqs enabled.

Note that this macro does not signify exclusivity. In fact multiple concurrent threads might go through. In that case external atomic (or locked) test on the condition might be necessary.
