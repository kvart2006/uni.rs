impl Packet

unsafe fn new(page: *mut u8, offset: usize, size: usize) -> Self

Creates a new packet from a memory pointer page

offset is the offset from the beginning of the page where the actual network data is

size is the size in bytes of the packet

This method is unsafe because a few requirements are necessary:

  • page must be allocated with __rust_allocate(PAGE_SIZE, PAGE_SIZE)
  • page ownership is transferred to this packet (i.e. it will be deallocated on Drop)
